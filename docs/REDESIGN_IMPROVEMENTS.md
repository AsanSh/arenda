# Список улучшений AMT Redesign (Mobile-First)

## 1. Навигация и структура
- ✅ Группировка меню по категориям (Финансы, Активы, Коммуникации, Система)
- ✅ Bottom Navigation для mobile (4-5 основных пунктов + "Еще")
- ✅ TopBar с поиском, уведомлениями и профилем
- ✅ Sticky header на всех страницах
- ✅ Safe-area поддержка для iOS

## 2. Адаптивность
- ✅ Mobile-first подход: сначала mobile, потом desktop
- ✅ Таблицы → карточки на mobile (<768px)
- ✅ Фильтры → bottom sheet на mobile
- ✅ Горизонтальный скролл KPI с snap на mobile
- ✅ Вкладки для списков на mobile (Просроченные/К оплате/Последние)

## 3. Компоненты и UI-kit
- ✅ Универсальный DataTable/DataCardList компонент (автопереключение)
- ✅ FilterBar (desktop) / FilterSheet (mobile bottom sheet)
- ✅ KPIStatCard с улучшенной визуализацией
- ✅ StatusChips для статусов (просрочено, к оплате)
- ✅ Amount компонент для единообразного форматирования денег
- ✅ Skeleton loading для всех состояний загрузки
- ✅ EmptyState и ErrorState компоненты

## 4. Производительность
- ✅ Виртуализация длинных списков (react-window или аналог)
- ✅ Lazy-load графиков
- ✅ Мемоизация дорогих вычислений
- ✅ Пагинация для больших данных
- ✅ Оптимизация перерисовок

## 5. UX улучшения
- ✅ Toast уведомления для действий
- ✅ Быстрые действия (FAB) на mobile для создания сущностей
- ✅ Swipe actions на карточках (опционально)
- ✅ Sticky summary на desktop, collapsible на mobile
- ✅ Предустановки периодов в Reports
- ✅ Мастер создания рассылки (wizard)

## 6. Доступность
- ✅ Минимальный размер клика 44px
- ✅ Контрастность текста (WCAG AA)
- ✅ ARIA labels для интерактивных элементов
- ✅ Фокус-индикаторы
- ✅ Keyboard navigation

## 7. Форматирование данных
- ✅ Единый формат денег: "100 000 сом" или "100 000 KGS"
- ✅ Единый формат дат: "25.01.2026"
- ✅ Intl.NumberFormat и Intl.DateTimeFormat
- ✅ Локализация для RU языка

## 8. Состояния и обратная связь
- ✅ Skeleton для KPI, таблиц, графиков
- ✅ EmptyState с CTA
- ✅ ErrorState с retry
- ✅ Loading states для всех асинхронных операций

## 9. Dashboard специфичные улучшения
- ✅ Горизонтальный скролл KPI с snap на mobile
- ✅ Вкладки для блоков (Просроченные/К оплате/Последние платежи)
- ✅ Переключатель графиков на mobile (Месяц/30 дней)
- ✅ Быстрые действия в PageHeader

## 10. Техническая архитектура
- ✅ Компонентная структура: /components/ui, /components/layout, /features/*
- ✅ Единый источник истины для фильтров (state + URL sync)
- ✅ Типизация всех сущностей
- ✅ Моковые данные для разработки
